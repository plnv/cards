# Витрина карточек

## Запуск
npx http-server -p 8000 
http://127.0.0.1:8000

## Возможности:
- Переключение между цитатами и задачами.
- Светлая и тёмная темы.
- Пагинация по 3 элемента на страницу.

## Технологии:
- HTML, CSS (custom properties), JavaScript (ES6)
- Web Components
- REST API https://dummyjson.com
